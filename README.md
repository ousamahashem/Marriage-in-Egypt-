# Marriage in Egypt - زواج الأجانب في مصر

هذا التطبيق يهدف لتسهيل خدمات زواج الأجانب في مصر، ويعتمد على تسجيل الدخول والدفع الآمن من خلال Pi Network.

## المميزات
- تسجيل دخول عبر Pi Browser
- تأكيد الهوية باستخدام Pi SDK
- خدمة قانونية موثوقة
